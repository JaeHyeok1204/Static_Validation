1. 동기화 제외 파일 설정 (.gitignore)
- node_modules/
- .env

2. 현재 작업 중인 컴퓨터에서 저장 및 업로드 (Push)
# 1. Git 저장소 초기화 (최초 1회)
git init

# 2. 모든 작업 파일 추가
git add .

# 3. 변경 사항 로컬에 저장
git commit -m "작업 내용 중간 저장"

# 4. GitHub 저장소 주소 연결 (GitHub에서 새 레포지토리를 만든 후 해당 URL 입력, 최초 1회)
git remote add origin <GitHub 레포지토리 URL>

# 5. 원격 저장소로 업로드
git push -u origin main

3. 다른 컴퓨터에서 작업 가져오기 및 세팅 (Clone & Pull)
# 1. GitHub에서 프로젝트 전체 다운로드
git clone <GitHub 레포지토리 URL>

# 2. 다운로드한 프로젝트 폴더로 진입
cd <프로젝트 폴더명>

# 3. 제외했던 패키지들을 다시 다운로드하여 개발 환경 복구
npm install

이후 추가로 작업한 내용만 가져올 때:
해당 컴퓨터에서 계속 작업을 이어나가려면 아래 명령어만 치면 최신 상태로 코드가 동기화됩니다.

git pull origin main
